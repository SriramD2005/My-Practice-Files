⚡ ReenuCircuit

🧠 Problem Description

Reenu is an electrical engineer who designs circuits using two types of resistors:

  • Horizontal resistor: - → allows current to flow left/right
  • Vertical resistor: | → allows current to flow up/down
  • Junction: + → connects all four directions with negligible resistance
  • Terminal: . → marks the opening and closing points of the circuit

The circuit is represented as an N × N matrix. Your task is to compute the total equivalent resistance between the opening and closing terminals, following valid paths and applying:

  • Series Rule: RT = R1 + R2
  • Parallel Rule: 1/RT = 1/R1 + 1/R2

📥 Input Format

  • First line: Integer N (size of the matrix), where 3 ≤ N ≤ 10
  • Next N lines: Each line contains N characters representing the circuit grid

📤 Output Format

A single integer representing the total resistance between the opening and closing terminals.

🧪 Symbols and Meaning

SymbolMeaning
.Terminal (start or end)
-Horizontal resistor (1Ω)
|Vertical resistor (1Ω)
+Junction (0Ω, connects all)

🧾 Example 1

Input:
4
.-+-
--|-
+-+-
|-+.

Output:
2

Explanation: The path from opening to closing passes through 2 resistors. Junctions add no resistance.

🧾 Example 2

Input:
5
.-+-+
--|-|
--|-|
--|-|
--+-.

Output:
3

Explanation: Two paths exist from opening to closing. Their resistances are reduced using parallel and series rules, resulting in a total of 3Ω.
